Song Pong

A Pong-Inspired Rhythm Game

About the Project

Song Pong is a stylistic and addicting vaporwave rhythm game inspired by Pong. Feel the groove and catch as many notes as you can. The demo includes four songs: 1999 She by Deaton Chris Anthony / Paradise by Coldplay / Midnight City by M83 / School by Bill Wurtz.

I took on this project with my friend and collaborator Sean Hruz. We wanted to challenge ourselves to make a game in a short time frame, and completed a first version of Song Pong in only 3 months.

Skills

I was the Artist and a Programmer, responsible for creating the rhythm logic and many other gameplay mechanics. I also created 3D graphics, such as the Paddle and the Buildings.

  • C#
  • Unity
  • Blender

itch.io

I've posted the demo on itch.io if you would like to try out Song Pong. There is currently a build for Mac and PC will be added soon.

Future Plans

Because it was created in a short time frame, there are many features that did not get completed. Here are some of things Song Pong has plans for in the future:

  • New types of Notes: Split and Slides
  • UI Overhaul
  • Visual Effects like Fog and reflective surfaces
  • Original Soundtrack
