Tomb of Khaibit

An Egyptian Co-op Horror Game

About the Project

Tomb of Khaibit is a co-op dungeon crawler where you play as archaeologists exploring a newly unearthed tomb in Egypt. In the dungeon, you'll discover horrifying creatures and magic from the Egyptian gods.

The tomb is procedurally generated and the puzzles you'll need to escape are randomized, meaning each tomb will be a new experience.

This project is for Senior Software Project with my group Blu3 J3ll0. You can see our group members below (we're pretty cool.)

Skills

Texo Editor

The Texo editor is what I am most proud of developing for this game. I spent around 60 hours creating it.

We needed a solution for Unity that let us procedurally generate our tomb. Unfortunately, Unity doesn't have any built-in Tools and the Asset Store has no solutions for a roguelike game. There was a package called Strata, which has since been removed from the Asset Store that provided a good basis. However, it was very limited in scope: (1) Rooms all needed to be the same size and (2) Exits needed to be located in certain places.

So I decided to expand on Strata and make my own editor called Texo. Texo uses arrows as connectors which allows for lots of flexibility, including rooms of various sizes and allowing exits to be placed anywhere. It allows us to generate tombs in many different formats.
